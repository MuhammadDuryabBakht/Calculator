 Calculator

 Overview

This is a simple **Calculator** application developed in **C++**. It is my first programming project and was created to practice the fundamentals of C++ programming. The program takes two numbers and an arithmetic operator as input from the user, then performs the selected operation using a `switch` statement and displays the result.

 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* Simple console-based user interface

 Technologies Used

* **Programming Language:** C++
* **Concepts Used:** Variables, Input/Output, `switch` statement, Arithmetic Operators, User Input

 How to Run

1. Compile the source code using any C++ compiler (e.g., g++, Code::Blocks, Dev-C++, or Visual Studio Code).
2. Run the executable.
3. Enter two numbers.
4. Choose an arithmetic operator (`+`, `-`, `*`, `/`).
5. View the calculated result.

 Learning Objectives

This project helped me understand the basics of C++ programming, including user input, decision-making with `switch` statements, arithmetic operations, and writing interactive console applications.

 Future Improvements
